Display a clean, customizable, and responsive mobile menu on your mobile site with quick links to the cart, user account, search or custom links.

== Description ==

Woosaleskit Bar add a menu with 3 icons to the footer of your mobile site for quick navigation.
The icons, links and styling can be set via the admin pane.

== Installation ==


1. Upload the ‘woosaleskit_bar’ folder to the ‘/wp-content/plugins/‘ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Do the setup via the admin menu

== Frequently Asked Questions ==

= Where will the bar show
The quick link bar will be fixed to the bottom of the screen and always visible

= Will the bar show on desktop
No, it will only show on mobile devices or devices with a screen resolution that is smaller that 768px
= Can I use custom link
Yes you can add your own links or use the predefined ones that link to the following Woocommerce links:
Cart page 
Account page
Product search

= Do I need Woocommerce installed for the plugin to work
No, if you don’t have Woocommerce installed then you can still use custom links. The Woocommerce links will not be available.
